std.logger.multilogger

class MultiLogger: std.logger.multilogger.MultiLoggerBase;
MultiLogger logs to multiple Logger.

It can be used to construct arbitrary, tree like structures. Basically a MultiLogger is a map. It maps Loggers to stringss. By adding a MultiLogger into another MultiLogger a non leaf nodes is added into the tree. The map is implemented as an associated array by the mapper MultiLogger.logger.

Example:
/+

root -> node -> b
|       |
|       |-> c
|-> a

+/

auto root = new MultiLogger(LogLevel.trace);
auto node = new MultiLogger(LogLevel.warning);

auto a = new FileLogger(stdout, LogLevel.trace);
auto b = new FileLogger(stdout, LogLevel.info);
auto c = new FileLogger(stdout, LogLevel.info);

root.insert("Node", node);
root.insert("a", a);

node.insert("b", b);
node.insert("c", c);

this(const LogLevel lv = LogLevel.info);
A constructor for the MultiLogger Logger.

Parameters:
LogLevel lv The LogLevel for the MultiLogger. By default the LogLevel for MultiLogger is LogLevel.info.

Example:
auto l1 = new MultiLogger(LogLevel.trace);

void insertLogger(string name, Logger newLogger);
This method inserts a new Logger into the MultiLogger.

Parameters:
string name The name of the Logger to insert. The name must be non empty and unique.
Logger newLogger The Logger to insert.

Examples:
auto l1 = new MultiLogger;
auto l2 = new FileLogger(stdout);

l1.insertLogger("someName", l2);

assert(l1.removeLogger("someName") is l2);

Logger removeLogger(string toRemove);
This method removes a Logger from the Multilogger.

Parameters:
string toRemove The name of the Logger to remove. If the Logger is not found an exception will be thrown.

Returns:
The removed Logger.

See Also:
std.logger.MultiLogger.insertLogger

Logger opIndex(string key);
This method returns a Logger if it is present in the MultiLogger, otherwise a RangeError will be thrown.

Examples:
auto ml = new MultiLogger;
auto sl = new FileLogger(stdout);

ml.insertLogger("some_name", sl);

assert(ml["some_name"] is sl);

class ArrayLogger: std.logger.multilogger.MultiLoggerBase;
ArrayLogger logs to multiple Logger. The Logger are stored in an std.container.Array in there order of insertion.

Every data logged to this ArrayLogger will be distributed to all the Logger inserted into inserted it. The ArrayLogger can hold multiple Logger with the same name. If the method removeLogger is used to remove a Logger only the first occurrence with that name will be removed.
